# Bevy 0.19.0 Local Docs

This project has a local, searchable copy of the Bevy 0.19.0 API documentation generated by `cargo doc`.

## Location

- `target/doc/bevy/index.html` — top-level Bevy crate docs.
- `target/doc/bevy_internal/index.html` — internal re-exports.
- `target/doc/<crate>/index.html` — docs for each sub-crate (e.g. `bevy_ecs`, `bevy_render`, `bevy_sprite`).

## How to open

```bash
# Open the top-level Bevy docs in your default browser
start target/doc/bevy/index.html
```

Or use the generated crate index:

```bash
start target/doc/k2c_clone/index.html
```

## How to search (from the terminal)

Use `ripgrep` against the generated HTML docs:

```bash
# Search for a type or function in the Bevy docs
rg -i "spawn_batch" target/doc/bevy

# Search all Bevy sub-crates
rg -i "Component" target/doc/bevy_*

# Search source snippets embedded in docs
rg -i "fn spawn" target/doc/src/bevy
```

For a broader search across the whole `target/doc` tree, use the same `rg` patterns.

## How to regenerate

If you add or remove Bevy features, or update the version, rebuild the docs:

```bash
cargo doc
```

To build only the local crate docs without dependencies:

```bash
cargo doc --no-deps
```

## Notes for this tutorial

- Keep these docs checked in? **No.** `target/` is gitignored and will be rebuilt locally.
- The first `cargo doc` run downloads and compiles all dependencies, so it can take a few minutes.
- For API questions, search the docs first, then look at `bevy/examples/` on GitHub if a usage pattern is unclear.
